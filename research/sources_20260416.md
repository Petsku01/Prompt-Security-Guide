## Key Themes Observed

1. **Defense-in-depth is the consensus**: No single defense solves prompt injection; layered mitigations are essential (PALADIN, tldrsec, CSA all agree).
2. **Multi-agent defense pipelines**: Emerging as a powerful approach (100% mitigation in arXiv:2509.14285).
3. **Agentic/MCP vulnerabilities are the new frontier**: Tool poisoning, unscoped API keys, and MCP client disparities are the operational threat surface.
4. **Multimodal attacks bypass text-only defenses**: Image-based prompt injection up to 64% success requires new detection approaches.
5. **RAG poisoning is highly effective**: Only 5 poisoned documents needed for 90% response manipulation.
6. **Prompt injection is architectural, not patchable**: Multiple sources frame it as fundamental; blast radius reduction > prevention.
