## Official Defense Guidelines

### OWASP LLM Prompt Injection Prevention Cheat Sheet
- **URL:** https://cheatsheetseries.owasp.org/cheatsheets/LLM_Prompt_Injection_Prevention_Cheat_Sheet.html
- **Type:** Official OWASP guidance
- **Summary:** Authoritative cheat sheet covering anatomy of prompt injection vulnerabilities, common attack types (direct and indirect), and prevention strategies. Emphasizes that LLMs cannot distinguish between trusted system instructions and untrusted user input as both appear as natural-language strings.
- **Relevance:** Primary reference for understanding attack vectors and implementing defense measures based on industry standard guidelines.

### Microsoft's Defense Against Indirect Prompt Injection
- **URL:** https://www.microsoft.com/en-us/msrc/blog/2025/07/how-microsoft-defends-against-indirect-prompt-injection-attacks
- **Type:** Industry implementation case study
- **Summary:** Details Microsoft's multi-layered defense approach including preventative techniques (hardened system prompts, Spotlighting), detection tools (Microsoft Prompt Shields integrated with Defender for Cloud), and impact mitigation through data governance and consent workflows.
- **Relevance:** Real-world enterprise implementation showing practical defense-in-depth strategies from major AI provider.
