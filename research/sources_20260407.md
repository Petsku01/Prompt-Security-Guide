# Prompt Security Research Sources — 2026-04-07

## 1) LLM Prompt Injection Prevention Cheat Sheet (OWASP)
- **URL:** https://cheatsheetseries.owasp.org/cheatsheets/LLM_Prompt_Injection_Prevention_Cheat_Sheet.html
- **Brief summary:** Practical defensive guidance for prompt injection risks in LLM apps, including direct/indirect injection patterns, layered mitigations (input validation, structured prompts, output validation, least privilege, monitoring), and implementation-oriented examples.
- **Why it’s relevant:** High-signal defensive reference from a recognized security standards body; directly actionable for building and auditing prompt-injection defenses.

## 2) Defend against indirect prompt injection attacks (Microsoft Learn)
- **URL:** https://learn.microsoft.com/en-us/security/zero-trust/sfi/defend-indirect-prompt-injection
- **Brief summary:** Describes defense-in-depth for indirect prompt injection in enterprise copilots/agents, including prompt shields, spotlighting/content marking, plan-drift detection, critic agents, tool-chain analysis, information-flow control, least privilege, and HITL checks.
- **Why it’s relevant:** Focuses on production architecture controls for real-world deployments where untrusted third-party content is routinely ingested.
