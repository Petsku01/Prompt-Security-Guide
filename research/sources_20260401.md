### Microsoft: How Microsoft Defends Against Indirect Prompt Injection
**URL:** https://www.microsoft.com/en-us/msrc/blog/2025/07/how-microsoft-defends-against-indirect-prompt-injection-attacks  
**Summary:** Microsoft's multi-layered defense approach including: hardened system prompts, Spotlighting technique to isolate untrusted inputs, Microsoft Prompt Shields detection, Defender for Cloud integration, data governance, user consent workflows, and deterministic blocking of data exfiltration methods.  
**Relevance:** Real-world implementation of defense-in-depth by major cloud provider.

### OWASP: LLM Prompt Injection Prevention Cheat Sheet

**Summary:** Official OWASP guidance on preventing prompt injection vulnerabilities in LLM applications. Defines prompt injection as top vulnerability (LLM01:2025) due to inability to distinguish between trusted system instructions and untrusted user input.  
**Relevance:** Authoritative standard for prompt injection prevention.
