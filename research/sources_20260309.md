## 2. OWASP Top 10 for LLM Applications 2025

**URL:** https://genai.owasp.org/llm-top-10/

**Summary:** The updated LLM Top 10 covering key risks:
- LLM01: Prompt Injection
- LLM02: Sensitive Information Disclosure
- LLM03: Supply Chain
- LLM04: Data and Model Poisoning
- LLM05: Improper Output Handling
- LLM06: Excessive Agency
- LLM07: System Prompt Leakage
- LLM08: Vector and Embedding Weaknesses
- LLM09: Misinformation
- LLM10: Unbounded Consumption

**Why Relevant:** Foundation document for LLM security. Prompt injection remains #1 risk. Detailed risk pages available for each category.

## 4. Model Context Protocol Security Project

**URL:** https://modelcontextprotocol-security.io/known-vulnerabilities/

**Summary:** Comprehensive security resource for MCP implementations. Tracks:
- Known CVEs and security advisories
- Real-world vulnerabilities in MCP systems
- TTP (Tactics, Techniques, Procedures) Matrix for MCP attacks

**Attack Categories Documented:**
- Prompt Injection & Manipulation (Direct, Indirect, Tool Description Poisoning, Context Shadowing)
- Tool Poisoning & Metadata Attacks (Tool Mutation, Name Conflicts, Shadowing)
- Data Exfiltration & Credential Theft
- Command & Code Injection
- Authentication & Authorization Bypass
- Supply Chain & Dependencies
- Context Manipulation
- Protocol Vulnerabilities
- Privilege & Access Control
- Economic & Infrastructure Abuse

**Why Relevant:** Most detailed MCP threat model available. Maintained by Cloud Security Alliance contributors. Critical for understanding emerging agentic attack surface.
