# Prompt Security Research Sources - 2026-03-11

## Academic Papers

### Prompt Injection Attacks on Large Language Models: A Survey
- **URL:** https://www.techscience.com/cmc/v87n1/66084
- **Authors:** Geng, Xu, Qu, Wong (State Information Center, Hohai University, UT Dallas)
- **Published:** 2026-02-10, Computers, Materials & Continua
- **Summary:** Systematic review synthesizing 128 peer-reviewed studies (2022-2025). Key findings: attacks achieve >90% success rates against unprotected systems; input preprocessing achieves 60-80% detection; advanced architectural defenses reach 95% protection against known patterns. Identifies 37 distinct defense approaches. Root causes attributed to fundamental LLM architectural limitations (can't distinguish instructions from data, attention mechanism vulnerabilities).
- **Relevance:** Comprehensive academic meta-analysis with quantified defense effectiveness. Essential reference for defense strategy benchmarking.
- **DOI:** https://doi.org/10.32604/cmc.2025.074081

## OWASP Resources

### LLM Prompt Injection Prevention Cheat Sheet (OWASP)
- **URL:** https://cheatsheetseries.owasp.org/cheatsheets/LLM_Prompt_Injection_Prevention_Cheat_Sheet.html
- **Summary:** Comprehensive attack taxonomy and defense patterns. Covers: direct injection, indirect/remote injection, encoding techniques (Base64, hex, Unicode), typoglycemia attacks, BoN jailbreaking, HTML/Markdown injection, multi-turn attacks, system prompt leakage, data exfiltration, multimodal injection, RAG poisoning, and agent-specific attacks. Includes Python code examples for sanitization filters.
- **Relevance:** Canonical reference - OWASP maintains this as the authoritative prevention guide. Updated with 2026 attack patterns.
